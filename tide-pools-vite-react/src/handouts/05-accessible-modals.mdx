# Exercise 05 — Accessible Modals  
⏱ ~30 minutes  
📖 EAA Category: Operable  

📂 Repo Reminder:

## 🛠 Hands-On (~20 minutes)  
- Add `role="dialog"` with `aria-labelledby` and `aria-describedby`.  
- Trap focus inside the modal.  
- Return focus to trigger on close.  
- Ensure Escape closes the modal.  

## ✅ Verify (~10 minutes)  
- Tab stays inside modal.  
- Screen reader announces title and description.  
- Focus returns to button when closed.
