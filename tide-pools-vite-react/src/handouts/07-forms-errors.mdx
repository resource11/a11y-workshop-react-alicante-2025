## 🛠 Hands-On (~15 minutes)  
- Add `<label>` for each input.  
- Associate error text with `aria-describedby`.  
- Add `aria-invalid="true"` on error.  
- Move focus to first invalid field.  
- Add an error summary at the top.  

## ✅ Verify (~10 minutes)  
- Errors announced by screen reader.  
- Keyboard focus moves to invalid input.  
- Summary announces how many errors exist.
