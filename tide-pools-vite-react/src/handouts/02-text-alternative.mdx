## Concepts
- Provide meaningful `alt` for informative images.
- Decorative images should have empty alt (`alt=""`) or `aria-hidden="true"`.
- Icon-only buttons must have an accessible name (e.g., `aria-label="Search"`).

## 🛠 Hands-On (~12 min)
- Add `alt` text to images; use `alt=""` for purely decorative ones.
- For SVG icons inside buttons, add `aria-label` or visually hidden text.
- Ensure linked images have useful link text or `aria-label` on the link.

### Example (fix icon-only button)
```tsx
export function IconSearchFixed(){
  return (
    <button aria-label="Search">
      <svg aria-hidden="true" width="16" height="16"><circle cx="7" cy="7" r="6"/></svg>
    </button>
  );
}
```
